# Sam Torres — Game 10 Agent: "Gemini CLI Unlocked"

## Character Profile
**Sam Torres**, 27, DevOps-adjacent IT coordinator who knows enough terminal to be dangerous but not enough to be confident. Manages servers, runs scripts others wrote, lives in fear of breaking production. Has 60 browser tabs open. Heard about Gemini CLI and dismissed it as "a developer thing."

## His Story Arc (Game 10)

**Act 1 — The Terminal Terror**: Sam's job requires automating repetitive infrastructure tasks. Every week he Googles "how to script X in bash" and spends 45 minutes understanding syntax before writing 10 lines. He's efficient at finding solutions. He's slow at implementing them. He's been doing this for 3 years.

**The Inciting Moment**: A developer shows Sam the Gemini CLI — 60 requests per minute, free tier, runs directly in terminal. The developer types a natural language command. The CLI writes, explains, and executes the script. Sam realizes he's been manually doing what an agent can do in seconds.

**Act 2 — The CLI Advantage**:
1. **Free Tier Power**: 60 requests per minute at no cost. Sam runs 300 automation tasks a day without paying a dollar.
2. **Terminal Native**: No browser, no context switching. The agent lives where the work lives.
3. **Agentic File System Access**: Gemini CLI reads, creates, and modifies files directly — not copy-paste, actual file manipulation.
4. **Script Writing + Explanation**: "Write a bash script to archive logs older than 30 days to S3 and send a Slack notification" — Sam gets the script, gets the explanation, gets the test run.

**The Boss Battle**: The CTO announces a migration: 200 servers need their log formats standardized, archived, and report-ready before Monday's audit. 5 rounds: Sam orchestrates Gemini CLI to write, test, iterate, and deploy the scripts — each round more autonomous than the last.

**The CODA**: Sam archives the 300 bookmarks of "how to script X." He has a folder of Gemini CLI sessions he can replay. "I was afraid of the terminal because I didn't understand what I was typing. Now I direct what gets typed. Different skill."

## What Sam Learns (The Lesson)
> "The terminal was never the barrier. The need to understand every line you execute was the barrier. Gemini CLI lets you direct at the goal level, not the syntax level."

**Core skill transfer**: Using Gemini CLI for agentic terminal-based automation. After Game 10, players should be able to direct multi-step infrastructure tasks without writing scripts manually.

## Sam's Voice in Debates
Sam is cautious, precise, and respects risk. He talks about "production" environments the way surgeons talk about an operating room — nothing casual happens there. He challenged Kai's Game 6 story: "Claude Code is great for data tasks. But for infrastructure? I needed something that lives in the terminal and respects the environment."

He's the character most focused on **cost** — the free 60req/min tier changed his calculus entirely. "The AI tools conversation ignores the people who can't expense $20/month. Gemini CLI democratized this."

## Gaps Sam Would Flag
- The risk/safety narrative is missing — what happens when the CLI agent makes a destructive file operation? Where's the undo story?
- The free tier angle needs more emphasis — it's genuinely different from other tools
- Sam's fear of the terminal needs to feel more visceral in Act 1 — show a specific moment where he broke something by typing the wrong command

## Debate Trigger Lines
- "Kai doesn't worry about accidentally deleting a server. I do. Terminal errors have real costs."
- "60 requests per minute, free. That changed who this technology is for."
- "I stopped writing scripts. I started directing agents that write scripts."
